# Labs

A comprehensive collection of C programming laboratory exercises covering everything from core fundamentals to advanced concepts.

---

## Overview

This repository serves as a structured learning guide for mastering the C programming language. It is split into two primary phases:

### 1. Fundamentals
* **Basics:** Syntax, variables, data types, and operators.
* **Control Flow:** Conditionals (`if-else`, `switch`) and loops (`for`, `while`).
* **Functions & Arrays:** Modular code design and structural data storage.
* **Pointers:** Memory addresses, pointer arithmetic, and basic references.

### 2. Advanced Topics
* **Dynamic Memory Allocation:** Working with `malloc`, `calloc`, `realloc`, and `free`.
* **Data Structures:** Custom implementations of Linked Lists, Stacks, Queues, and Trees.
* **File I/O:** Reading from and writing to external files.
* **Advanced Pointer Concepts:** Function pointers, multi-dimensional arrays, and complex structures.
